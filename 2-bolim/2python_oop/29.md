## 1. Kirish va Class/Object tushunchasi (20 daqiqa)

Assalomu alaykum, yosh dasturchilar! üòä Bugun biz Python‚Äôda Class (Sinf) va Object (Obyekt) tushunchasini chuqurroq o‚Äòrganamiz. Ayniqsa, class atributlari, instance atributlari va `__init__` metodi haqida gaplashamiz.

**Class va Object nima?**  
- **Class (Sinf)**: Obyektlar uchun shablon yoki qolip. Masalan, "Mashina" sinfi mashinalar uchun umumiy xususiyatlarni belgilaydi.  
- **Object (Obyekt)**: Sinfdan yaratilgan konkret nusxa. Masalan, "Mashina" sinfidan "Chevrolet" obyekti yaratiladi.  

Class va object yordamida kodni tartibli va qayta ishlatiladigan qilish mumkin. Bugun biz sinf ichida atributlarni qanday ishlatishni va `__init__` metodining vazifasini o‚Äòrganamiz.

Oddiy misol bilan boshlaymiz:

```python
class Talaba:
    def __init__(self, ism):
        self.ism = ism

talaba1 = Talaba("Ali")
print(talaba1.ism)  # Ali
```

Bu yerda `Talaba` ‚Äî sinf, `talaba1` ‚Äî obyekt, `ism` esa obyektning atributi.

Mashq: Quyidagi vazifani bajaring:  
- `Mashina` nomli sinf yarating, u `model` atributiga ega bo‚Äòlsin.  
- Bitta obyekt yarating va uning modelini ekranga chiqaring.

## 2. Class atributlari va instance atributlari (25 daqiqa)

Endi class atributlari va instance atributlarini o‚Äòrganamiz! Atributlar sinf yoki obyektning xususiyatlarini ifodalaydi.

### Class atributlari
Class atributlari sinfning barcha obyektlari uchun umumiy bo‚Äòladi. Ular sinf ichida, `__init__` metodidan tashqarida e‚Äôlon qilinadi:

```python
class Universitet:
    # Class atributi
    nom = "TATU"

    def __init__(self, talaba_ism):
        self.talaba_ism = talaba_ism

talaba1 = Universitet("Sardor")
talaba2 = Universitet("Nodira")

print(f"{talaba1.talaba_ism} {Universitet.nom}da o‚Äòqiydi.")  # Sardor TATUda o‚Äòqiydi.
print(f"{talaba2.talaba_ism} {Universitet.nom}da o‚Äòqiydi.")  # Nodira TATUda o‚Äòqiydi.
```

Bu yerda `nom` ‚Äî class atributi bo‚Äòlib, barcha obyektlar uchun bir xil.

### Instance atributlari
Instance atributlari har bir obyekt uchun alohida bo‚Äòladi va odatda `__init__` metodida e‚Äôlon qilinadi:

```python
class Hayvon:
    def __init__(self, turi, yosh):
        # Instance atributlari
        self.turi = turi
        self.yosh = yosh

hayvon1 = Hayvon("Mushuk", 3)
hayvon2 = Hayvon("Kuchuk", 5)

print(f"{hayvon1.turi} {hayvon1.yosh} yoshda.")  # Mushuk 3 yoshda.
print(f"{hayvon2.turi} {hayvon2.yosh} yoshda.")  # Kuchuk 5 yoshda.
```

### Class va instance atributlari o‚Äòrtasidagi farq
Class atributlari umumiy bo‚Äòlsa, instance atributlari har bir obyekt uchun o‚Äòziga xos bo‚Äòladi:

```python
class Do‚Äòkon:
    # Class atributi
    turi = "Oziq-ovqat"

    def __init__(self, nomi, mahsulotlar_soni):
        # Instance atributlari
        self.nomi = nomi
        self.mahsulotlar_soni = mahsulotlar_soni

dokon1 = Do‚Äòkon("Alisher", 50)
dokon2 = Do‚Äòkon("Nodir", 30)

print(f"{dokon1.nomi} do‚Äòkoni ({Do‚Äòkon.turi}), mahsulotlar soni: {dokon1.mahsulotlar_soni}")  # Alisher do‚Äòkoni (Oziq-ovqat), mahsulotlar soni: 50
print(f"{dokon2.nomi} do‚Äòkoni ({Do‚Äòkon.turi}), mahsulotlar soni: {dokon2.mahsulotlar_soni}")  # Nodir do‚Äòkoni (Oziq-ovqat), mahsulotlar soni: 30
```

Class atributlari va instance atributlari yordamida obyektlarning umumiy va o‚Äòziga xos xususiyatlarini boshqarish mumkin.

Mashq: Quyidagi vazifani bajaring:  
- `Kitobxona` nomli sinf yarating, u `turi` nomli class atributiga (qiymati: "Umumiy") va `nomi`, `kitoblar_soni` nomli instance atributlariga ega bo‚Äòlsin.  
- Ikkita kitobxona obyekti yarating va ularning ma‚Äôlumotlarini ekranga chiqaring.

## 3. __init__ metodi bilan ishlash (20 daqiqa)

Ajoyib, endi `__init__` metodi haqida chuqurroq gaplashamiz! `__init__` metodi sinfning **konstruktori** deb ataladi va obyekt yaratilganda avtomatik chaqiriladi.

### __init__ metodining vazifasi
`__init__` metodi obyektning boshlang‚Äòich holatini sozlash uchun ishlatiladi. U instance atributlarini boshlang‚Äòich qiymat bilan ta‚Äôminlaydi:

```python
class Talaba:
    def __init__(self, ism, yosh):
        self.ism = ism
        self.yosh = yosh

    def malumot(self):
        return f"{self.ism} {self.yosh} yoshda."

talaba = Talaba("Ali", 20)
print(talaba.malumot())  # Ali 20 yoshda.
```

Bu yerda `__init__` metodi `ism` va `yosh` atributlarini boshlang‚Äòich qiymat bilan sozlaydi.

### Parametrsiz __init__ metodi
`__init__` metodida faqat `self` parametri bo‚Äòlishi ham mumkin:

```python
class Mashina:
    def __init__(self):
        self.rang = "Qora"  # Standart qiymat
        self.tezlik = 0

mashina = Mashina()
print(f"Rang: {mashina.rang}, Tezlik: {mashina.tezlik} km/soat")  # Rang: Qora, Tezlik: 0 km/soat
```

### Class atributlari bilan birga ishlatish
`__init__` metodida class atributlarini ham ishlatish mumkin:

```python
class Universitet:
    nom = "TATU"

    def __init__(self, talaba_ism, kurs):
        self.talaba_ism = talaba_ism
        self.kurs = kurs

    def malumot(self):
        return f"{self.talaba_ism} {Universitet.nom}ning {self.kurs}-kurs talabasi."

talaba = Universitet("Zilola", 2)
print(talaba.malumot())  # Zilola TATUning 2-kurs talabasi.
```

`__init__` metodi obyektni moslashtirishda juda muhim rol o‚Äòynaydi.

Mashq: Quyidagi vazifani bajaring:  
- `Telefon` nomli sinf yarating, u `__init__` metodi orqali `marka` va `narx` atributlarini sozlasin.  
- `malumot` nomli metod yozing, u telefon haqida ma‚Äôlumot qaytarsin.  
- Bitta telefon obyekti yarating va ma‚Äôlumotni ekranga chiqaring.  

5 daqiqa vaqt beraman!

## 4. Amaliy mashqlar (25 daqiqa)

### Mashq 1: Class va instance atributlari
- `Maktab` nomli sinf yarating, u `turi` nomli class atributiga (qiymati: "Umumiy ta‚Äôlim") va `nomi`, `oquvchilar_soni` nomli instance atributlariga ega bo‚Äòlsin.

### Mashq 2: __init__ metodi
- `Kitob` nomli sinf yarating, u `__init__` metodi orqali `nomi` va `muallif` atributlarini sozlasin.  
- `malumot` nomli metod yozing, u kitob haqida ma‚Äôlumot qaytarsin.  
- Ikkita kitob obyekti yarating va ma‚Äôlumotlarini ekranga chiqaring.

```python

    def __init__(self, nomi, muallif):
        self.nomi = nomi

kitob2 = Kitob("O‚Äòtgan kunlar", "Abdulla Qodiriy")
```

### Mashq 3: Atributlarni o‚Äòzgartirish
- `Mashina` nomli sinf yarating, u `__init__` metodi orqali `model` va `tezlik` atributlarini sozlasin.  
- `tezlik_oshir` nomli metod yozing, u tezlikni 20 ga oshirsin.  
- Bitta mashina obyekti yarating, tezlikni oshirib, yangi tezlikni ekranga chiqaring.

```python

    def __init__(self, model, tezlik):
        self.model = model
        self.tezlik = tezlik

    def tezlik_oshir(self):
        self.tezlik += 20
        return self.tezlik

print(f"Yangi tezlik: {mashina.tezlik_oshir()} km/soat")  # Yangi tezlik: 100 km/soat
```
