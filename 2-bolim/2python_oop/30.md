## 1. Kirish va metodlar tushunchasi (20 daqiqa)

Assalomu alaykum, yosh dasturchilar! üòä Bugun biz Python‚Äôda OOP (Obyektga yo‚Äònaltirilgan dasturlash) doirasida metodlar haqida o‚Äòrganamiz. Ayniqsa, instance metodlar, class metodlar va static metodlar o‚Äòrtasidagi farqlarni ko‚Äòrib chiqamiz.

**Metod nima?**  
Metod ‚Äî bu sinf ichida e‚Äôlon qilinadigan funksiya bo‚Äòlib, obyekt yoki sinf bilan bog‚Äòliq harakatlarni bajaradi. Python‚Äôda metodlarning 3 asosiy turi mavjud:  
- **Instance metodlar**: Obyekt (instance) bilan ishlaydi.  
- **Class metodlar**: Sinf bilan ishlaydi.  
- **Static metodlar**: Sinf yoki obyektga bog‚Äòliq bo‚Äòlmagan mustaqil metodlar.  

Oddiy misol bilan boshlaymiz:

```python
class Talaba:
    def __init__(self, ism):
        self.ism = ism

    # Instance metod
    def salom_ber(self):
        return f"Salom, men {self.ism}!"

talaba = Talaba("Ali")
print(talaba.salom_ber())  # Salom, men Ali!
```

Bu yerda `salom_ber` ‚Äî instance metod, chunki u obyekt (`talaba`) bilan ishlaydi.

## 2. Instance metodlar bilan ishlash (25 daqiqa)

Endi instance metodlarni chuqurroq o‚Äòrganamiz! Instance metodlar obyekt bilan ishlaydi va odatda `self` parametrini qabul qiladi.

### Instance metod nima?
Instance metod ‚Äî bu sinf ichida e‚Äôlon qilinadigan va obyektning atributlari bilan ishlaydigan metod. U `self` orqali obyektning ma‚Äôlumotlariga kiradi:

```python
class Kitob:
    def __init__(self, nomi, narxi):
        self.nomi = nomi
        self.narxi = narxi

    # Instance metod
    def malumot(self):
        return f"Kitob: {self.nomi}, Narxi: {self.narxi} so‚Äòm"

    # Instance metod (atributni o‚Äòzgartirish)
    def narx_oshir(self, oshirish):
        self.narxi += oshirish
        return f"Yangi narx: {self.narxi} so‚Äòm"

kitob = Kitob("Alpomish", 50000)
print(kitob.malumot())  # Kitob: Alpomish, Narxi: 50000 so‚Äòm
print(kitob.narx_oshir(10000))  # Yangi narx: 60000 so‚Äòm
```

### Instance metodning xususiyatlari
- `self` parametri orqali obyektning atributlariga kiradi.  
- Faqat obyekt orqali chaqiriladi (`kitob.malumot()`).  
- Obyektning holatini o‚Äòzgartirishi mumkin (`narx_oshir` kabi).

### Ko‚Äòproq misol
Instance metod yordamida obyektning holatini boshqarish:

```python
class Talaba:
    def __init__(self, ism, baho):
        self.ism = ism
        self.baho = baho

    def baho_korsat(self):
        return f"{self.ism}ning bahosi: {self.baho}"

    def baho_ozgartir(self, yangi_baho):
        self.baho = yangi_baho
        return f"{self.ism}ning yangi bahosi: {self.baho}"

talaba = Talaba("Zilola", 85)
print(talaba.baho_korsat())  # Zilolaning bahosi: 85
print(talaba.baho_ozgartir(90))  # Zilolaning yangi bahosi: 90
```

Instance metodlar obyektga xos harakatlarni bajarishda juda muhim.

Mashq: Quyidagi vazifani bajaring:  
- `Mashina` nomli sinf yarating, u `model` va `tezlik` atributlariga ega bo‚Äòlsin.  
- `tezlik_oshir` nomli instance metod yozing, u tezlikni 10 ga oshirsin.  
- Bitta mashina obyekti yarating, tezlikni oshirib, natijani ekranga chiqaring.

## 3. Class va Static metodlar (20 daqiqa)

Ajoyib, endi class metodlar va static metodlarni o‚Äòrganamiz! Bu metodlar instance metodlardan farqli ravishda sinf yoki umumiy maqsadlar uchun ishlatiladi.

### Class metodlar
Class metodlar sinf bilan ishlaydi va `@classmethod` dekoratori yordamida e‚Äôlon qilinadi. U `cls` parametrini qabul qiladi:

```python
class Universitet:
    nom = "TATU"

    def __init__(self, talaba_ism):
        self.talaba_ism = talaba_ism

    @classmethod
    def nom_ozgartir(cls, yangi_nom):
        cls.nom = yangi_nom
        return f"Yangi nom: {cls.nom}"

talaba = Universitet("Sardor")
print(Universitet.nom)  # TATU
print(Universitet.nom_ozgartir("TDTU"))  # Yangi nom: TDTU
print(Universitet.nom)  # TDTU
```

Class metodlar sinfning umumiy atributlarini o‚Äòzgartirishda foydali. Ular sinf orqali chaqiriladi (`Universitet.nom_ozgartir()`).

### Static metodlar
Static metodlar sinf yoki obyektga bog‚Äòliq bo‚Äòlmagan mustaqil metodlardir. Ular `@staticmethod` dekoratori yordamida e‚Äôlon qilinadi:

```python
class Kalkulyator:
    @staticmethod
    def yigindi(a, b):
        return a + b

    @staticmethod
    def ayirma(a, b):
        return a - b

print(Kalkulyator.yigindi(5, 3))  # 8
print(Kalkulyator.ayirma(10, 4))  # 6
```

Static metodlar:  
- `self` yoki `cls` parametrlarini talab qilmaydi.  
- Sinf bilan bog‚Äòliq bo‚Äòlmagan umumiy vazifalar uchun ishlatiladi.  
- Sinf orqali chaqiriladi (`Kalkulyator.yigindi()`).

### Class va Static metodlar o‚Äòrtasidagi farq
Class metodlar sinfning atributlari bilan ishlasa, static metodlar sinfga umuman bog‚Äòliq emas:

```python
class Do‚Äòkon:
    turi = "Oziq-ovqat"

    @classmethod
    def turi_ozgartir(cls, yangi_turi):
        cls.turi = yangi_turi
        return f"Yangi turi: {cls.turi}"

    @staticmethod
    def chegirma_hisobla(narx, foiz):
        return narx * (1 - foiz / 100)

print(Do‚Äòkon.turi)  # Oziq-ovqat
print(Do‚Äòkon.turi_ozgartir("Kiyim"))  # Yangi turi: Kiyim
print(Do‚Äòkon.chegirma_hisobla(100000, 20))  # 80000.0
```

Class va static metodlar sinf darajasida ishlaydi va turli maqsadlar uchun foydalidir.

Mashq: Quyidagi vazifani bajaring:  
- `Kitobxona` nomli sinf yarating, u `kitoblar_soni` nomli class atributiga ega bo‚Äòlsin.  
- `kitoblar_sonini_oshir` nomli class metod yozing, u kitoblar sonini oshirsin.  
- `narx_hisobla` nomli static metod yozing, u kitob narxini (masalan, 1 kitob 5000 so‚Äòm) hisoblasin.  
- Class va static metodlarni ishlatib natijalarni ekranga chiqaring.
