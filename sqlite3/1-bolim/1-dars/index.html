🗄️ SQLite nima? - Ma'lumotlar bazasi tushunchasi

SQLite nima?

SQLite - bu eng mashhur ma'lumotlar bazasi muhandisligi (database engine) hisoblanadi. U butun dunyoda milliardlab qurilmalarda ishlatiladi.

SQLite xususiyatlari:

  • Server-siz: SQLite server talab qilmaydi
  • Zero-configuration: O'rnatish va sozlash talab qilmaydi
  • Cross-platform: Barcha operatsion tizimlarda ishlaydi
  • Self-contained: Bitta fayl sifatida saqlanadi
  • ACID-compliant: Ma'lumotlar yaxlitligini ta'minlaydi

Ma'lumotlar bazasi nima?

Ma'lumotlar bazasi - bu ma'lumotlarni tizimli ravishda saqlash, boshqarish va qidirish uchun ishlatiladigan tizim.

SQLite boshqa DBMSlardan farqi:

Xususiyat SQLite MySQL PostgreSQL
Server Yo'q Ha Ha
O'rnatish Oddiy Murakkab Murakkab
Fayl hajmi Kichik Katta Katta
Ishlatish Mobile, Desktop Web Enterprise

SQLite qayerda ishlatiladi?

  • 📱 Mobile ilovalar (Android, iOS)
  • 💻 Desktop dasturlar
  • 🌐 Web ilovalar
  • 🔧 Embedded tizimlar
  • 📊 Prototyping va testing

SQLite afzalliklari:

  • ✅ Tez o'rnatish
  • ✅ Server talab qilmaydi
  • ✅ Bitta fayl
  • ✅ Kichik hajm
  • ✅ Tez ishlash
  • ✅ Xavfsiz

SQLite cheklovlari:

  • ❌ Bir vaqtda ko'p foydalanuvchi
  • ❌ Network orqali ulanish
  • ❌ Katta hajmdagi ma'lumotlar
  • ❌ Murakkab concurrent operations

Misol:

SQLite fayl yaratish:

sqlite3 mydatabase.db

Bu buyruq yangi ma'lumotlar bazasi faylini yaratadi.
