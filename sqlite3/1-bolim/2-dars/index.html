Windows da o'rnatish

1-usul: Ready-to-use binary

  1. SQLite.org saytiga o'ting
  2. "Precompiled Binaries for Windows" bo'limini toping
  3. sqlite-tools-win32-x86-*.zip faylini yuklab oling
  4. Arxivni oching va sqlite3.exe ni C:\sqlite\ papkasiga ko'chiring
  5. Environment Variables ga C:\sqlite\ ni qo'shing

2-usul: Chocolatey orqali

# Chocolatey o'rnatilgan bo'lsa
choco install sqlite3

# Tekshirish
sqlite3 --version

3-usul: Scoop orqali

# Scoop o'rnatilgan bo'lsa
scoop install sqlite3

# Tekshirish
sqlite3 --version

O'rnatishni tekshirish

# SQLite versiyasini ko'rish
sqlite3 --version

# SQLite CLI ni ishga tushirish
sqlite3

# Chiqish
.quit

Python bilan SQLite

Python da SQLite3 moduli standart kutubxona hisoblanadi:

# Python da SQLite3 modulini import qilish
import sqlite3

# Versiyani tekshirish
print(sqlite3.sqlite_version)

Node.js bilan SQLite

# npm orqali sqlite3 paketini o'rnatish
npm install sqlite3

# Yoki better-sqlite3 (tezroq)
npm install better-sqlite3

Muammolarni hal qilish

Linux da muammolar:

  • Package manager yangilang: sudo apt update
  • Permissions muammosi: sudo chmod +x /usr/bin/sqlite3

Windows da muammolar:

  • PATH environment variable ni tekshiring
  • Command Prompt ni restart qiling
  • Administrator sifatida ishga tushiring

macOS da muammolar:

  • Xcode Command Line Tools o'rnating: xcode-select --install
  • Homebrew o'rnating: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
