⚡ Asosiy buyruqlar - .open, .tables, .schema, .help

SQLite CLI buyruqlari

SQLite CLI da nuqta (.) bilan boshlanadigan buyruqlar "dot commands" deb ataladi.

.help - Yordam

sqlite3
.help

Bu buyruq barcha mavjud dot commands ro'yxatini ko'rsatadi.

.open - Ma'lumotlar bazasini ochish

# Yangi ma'lumotlar bazasi yaratish
.open mydatabase.db

# Mavjud ma'lumotlar bazasini ochish
.open existing_database.db

# Xotira bazasini ochish (vaqtinchalik)
.open :memory:

.tables - Jadvallarni ko'rish

# Barcha jadvallarni ko'rish
.tables

# Belgili pattern bo'yicha qidirish
.tables user*
.tables *log*

.schema - Jadval strukturasi

# Barcha jadvallar strukturasi
.schema

# Belgili jadval strukturasi
.schema users
.schema products

# Indekslarni ham ko'rish
.schema --indices

.quit / .exit - Chiqish

.quit
# yoki
.exit

Boshqa foydali buyruqlar

.databases - Ma'lumotlar bazalarini ko'rish

.databases

.headers - Sarlavhalarni ko'rsatish

# Sarlavhalarni yoqish
.headers on

# Sarlavhalarni o'chirish
.headers off

.mode - Natija formati

# Jadval formatida
.mode table

# CSV formatida
.mode csv

# HTML formatida
.mode html

# JSON formatida
.mode json

# List formatida
.mode list

.output - Natijani faylga saqlash

# Natijani faylga saqlash
.output results.txt
SELECT * FROM users;
.output stdout

# Ekranga qaytarish
.output stdout

.show - Hozirgi sozlamalarni ko'rish

.show

Amaliy misol

# SQLite ni ishga tushirish
sqlite3

# Ma'lumotlar bazasini ochish
.open test.db

# Sarlavhalarni yoqish
.headers on

# Jadval formatini tanlash
.mode table

# Yordam olish
.help

# Ma'lumotlar bazasini yaratish
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    email TEXT UNIQUE
);

# Jadvallarni ko'rish
.tables

# Jadval strukturasi
.schema users

# Ma'lumot qo'shish
INSERT INTO users (name, email) VALUES ('Ali', 'ali@example.com');

# Ma'lumotlarni ko'rish
SELECT * FROM users;

# Chiqish
.quit
